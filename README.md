# Multi-Document Financial Intelligence Assistant with Agentic AI

## Business Problem

Financial analysts spend hours reviewing annual reports, quarterly filings, earnings call transcripts, and investor presentations to answer business questions.

This project automates that workflow using Retrieval-Augmented Generation (RAG) and a multi-agent architecture built with LangGraph.

## Features

- Automated SEC data ingestion
- PDF parsing
- Semantic chunking
- FAISS vector search
- LangGraph agent workflow
- Financial reasoning
- FastAPI backend
- Streamlit frontend
- Docker deployment
