# Multi-Document Financial Intelligence Assistant with Agentic AI

## Overview

An enterprise-grade AI platform that automates financial document analysis using Retrieval-Augmented Generation (RAG) and Agentic AI.

The platform ingests SEC filings, investor presentations, and earnings call transcripts to answer complex financial questions with explainable citations.

## Business Problem

Financial analysts spend hours reviewing annual reports, quarterly filings, earnings transcripts, and investor presentations to prepare executive summaries, compare companies, and identify risks.

This project automates that workflow using AI.

## Features

- Automated SEC EDGAR data ingestion
- PDF and HTML document parsing
- Semantic chunking
- Vector search using FAISS
- Multi-Agent workflow using LangGraph
- Financial KPI extraction
- Executive report generation
- FastAPI backend
- Streamlit frontend
- Docker deployment
- AWS deployment
